# Bíblias em Português

Coloque aqui os arquivos JSON das versões bíblicas em português.

## Versões Disponíveis:
- **NVI** - Nova Versão Internacional
- **ACF** - Almeida Corrigida Fiel
- **AA** - Almeida Atualizada

## Formato do Arquivo:
Os arquivos devem estar no formato JSON compatível com o sistema.

Exemplo de estrutura:
```json
[
  {
    "abbrev": "gn",
    "chapters": [...],
    "name": "Gênesis"
  }
]
```
