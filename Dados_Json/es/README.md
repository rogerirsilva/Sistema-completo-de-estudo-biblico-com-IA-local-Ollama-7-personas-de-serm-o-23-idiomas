# Biblias en Español

Coloque aquí los archivos JSON de las versiones bíblicas en español.

## Versiones Recomendadas:
- **RV1960** - Reina Valera 1960
- **NVI** - Nueva Versión Internacional
- **RVA** - Reina Valera Antigua

## Formato de Archivo:
Los archivos deben estar en formato JSON compatible con el sistema.
